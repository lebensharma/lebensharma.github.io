Privacy Policy for Hover Currency Converter

Hover Currency Converter detects visible currency amounts on webpages and shows converted values on hover. The extension processes page text locally in the browser. It does not collect accounts, passwords, form entries, or browsing history.

Information We Process

The extension reads visible text on webpages in order to detect currency amounts and display conversion tooltips. This text processing happens locally in the browser.

Information We Store

The extension stores exchange-rate cache data and related metadata locally using Chrome storage so conversions can load faster and repeated API requests can be reduced.

Information We Do Not Collect

The extension does not intentionally collect personal information, account credentials, payment details, form input, messages, or browsing history. It does not sell user data.

Remote Requests

The extension sends a request to the configured exchange-rate API to retrieve currency conversion rates. These requests are used only to fetch rate data. Page text and detected currency matches are not sent to the API by the extension.

Host Permissions

The extension requests access to webpage content so it can detect visible currency amounts directly on visited pages and show hover-based conversions.

Data Sharing

The extension does not share user data with third parties except for the network request needed to retrieve exchange-rate data from the configured rate provider.

Data Retention

Locally cached exchange-rate data remains in browser storage until it is refreshed, overwritten, or removed by the user or browser.

Security

The extension is designed to minimize data use by processing webpage text locally and storing only limited rate-cache information needed for its functionality.

Third-Party Services

If you configure a third-party exchange-rate provider, that provider may receive standard network information such as IP address and request metadata as part of the rate request. Their handling of that information is governed by their own privacy policy.

Changes to This Policy

This privacy policy may be updated if the extension's behavior or data practices change.
